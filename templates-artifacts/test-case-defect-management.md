# 📅 Description

This document contains **supporting templates for managing test cases and defects**, designed to facilitate tracking, control, and traceability of Quality Assurance (QA) activities.

The included formats allow teams to:

- Document test cases
- Record execution results
- Report defects identified during testing

Using these templates helps maintain a **standardized structure and a clear view of the product's quality status** throughout the testing lifecycle.

# 🗒️ Notes

Although the templates provided can be used to track and document testing activities, it is recommended to use **test management tools** such as:

- Jira
- Zephyr Scale
- Xray

These platforms allow teams to:

- Centralize testing information
- Link test cases with defects
- Generate automated reports
- Improve visibility of testing progress and product quality
